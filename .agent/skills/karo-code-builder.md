# Karo Code Builder

## Mục đích
Đây là kỹ năng nòng cốt giúp xây dựng các thành phần mã nguồn từ logic đến giao diện, đảm bảo tính nhất quán và chất lượng cao nhất.

## Hướng dẫn thực hiện
1.  **Phân tích Spec:** Đọc kỹ tài liệu yêu cầu (Spec) trước khi bắt đầu viết code.
2.  **Thiết lập cấu trúc:** Xác định vị trí tệp tin phù hợp trong project (`src/components`, `src/hooks`, vv.).
3.  **Xây dựng Logic:** Viết code xử lý logic trước, đảm bảo tính đúng đắn.
4.  **Thiết kế Giao diện:** Áp dụng `design-rules.md` để tạo UI premium với TailwindCSS hoặc CSS vanilla.
5.  **Tối ưu hóa:** Loại bỏ code dư thừa, tối ưu hóa các lần render.

## Tiêu chí chất lượng
- Code phải tuân thủ `coding-rules.md`.
- Giao diện phải responsive và có hiệu ứng mượt mà.
- Phải có xử lý các trạng thái: Loading, Error, Empty.
