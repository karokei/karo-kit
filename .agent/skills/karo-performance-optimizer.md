# Karo Performance Optimizer

## Mục đích
Kỹ năng tối ưu hóa sâu (Deep Optimization), giải quyết các bài toán về tốc độ và quy mô hệ thống.

## Phạm vi tối ưu
- **Frontend Perf:** Tối ưu hóa render tree, lazy loading, nén ảnh, giảm thiểu script của bên thứ ba.
- **Backend Perf:** Tối ưu hóa cấu trúc dữ liệu, các thuật toán xử lý lớn, sử dụng caching (Redis/Memcached).
- **Database Perf:** Tối ưu hóa câu truy vấn, sử dụng Index, xem xét Partitioning khi cần.
- **Network Perf:** Sử dụng CDN, nén Gzip/Brotli, tối ưu hóa giao thức HTTP/2, HTTP/3.

## Chỉ dẫn hiệu năng
- Thu thập dữ liệu đo lường trước khi thực hiện.
- Luôn kiểm tra trong môi trường sát với thực tế (Staging).
