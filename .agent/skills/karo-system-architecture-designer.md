# Karo System Architecture Designer

## Mục đích
Thiết kế kiến trúc hệ thống tổng thể, đảm bảo tính bền vững, khả năng chịu tải và dễ dàng phát triển.

## Nội dung thiết kế
- **Mô hình kiến trúc:** Microservices, Monolith, Serverless, vv. phù hợp với quy mô dự án.
- **Lựa chọn Tech Stack:** Đề xuất ngôn ngữ, framework và database phù hợp nhất.
- **Sơ đồ tích hợp:** Cách thức các thành phần trong hệ thống giao tiếp với nhau (API, Message Queue).
- **Khả năng chịu lỗi:** Thiết kế cơ chế dự phòng và tự phục hồi (High Availability).

## Đầu ra
- Tài liệu kiến trúc chuyên sâu.
- Các sơ đồ Mermaid chi tiết.
