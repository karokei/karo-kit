# product-analyzer

## Purpose

Skill này dùng để thực hiện nhiệm vụ chuyên môn liên quan đến **product-analyzer**.

## Scope

Áp dụng trong các workflow:

- analyze-product
- generate-spec
- build-system
- optimize-performance
- fix-bug

## Input

- system context
- user request
- project data
- previous outputs

## Output

- structured result
- analysis/spec/code/documentation

## Process

1. đọc input
2. phân tích yêu cầu
3. áp dụng logic của skill
4. tạo output
5. validate
6. trả kết quả

## Rules

Follow:

.agent/rules/engineering-rules.md  
.agent/rules/ai-rules.md  
.agent/rules/security-rules.md  
.agent/rules/workflow-rules.md  

## Example

Input:

phân tích hệ thống SaaS

Output:

phân tích hoàn chỉnh

## Anti-patterns

- bỏ qua rules
- tạo output không cấu trúc
- hallucinate
