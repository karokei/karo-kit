# Karo Workflow: Develop Feature

## Giới thiệu
Quy trình chuẩn để phát triển một tính năng mới từ ý tưởng đến khi hoàn thiện mã nguồn.

## Các bước thực hiện
1.  **Nghiên cứu yêu cầu:** Gọi `karo-analyst` để phân tích nhu cầu của sếp.
2.  **Thiết kế Spec:** Tạo tài liệu đặc tả (Spec) chi tiết về logic và UI.
3.  **Lập kế hoạch:** Tạo `implementation_plan.md` và `task.md`.
4.  **Xây dựng nền tảng:** Tạo cấu trúc thư mục và các tệp tin cần thiết.
5.  **Thực hiện mã hóa:** Gọi `karo-ai-engineer` để viết code theo từng task.
6.  **Kiểm soát chất lượng:** Chạy test và review lại code.
7.  **Bàn giao:** Tạo `walkthrough.md` để sếp kiểm tra kết quả.

## Lệnh kích hoạt ví dụ
- "Sếp muốn thêm tính năng giỏ hàng cho web."
- "Tạo trang dashboard mới sếp ơi."
