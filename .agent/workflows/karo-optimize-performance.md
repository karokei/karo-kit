# Karo Workflow: Optimize Performance

## Giới thiệu
Quy trình tối ưu hóa tốc độ và trải nghiệm người dùng một cách chuyên sâu.

## Các bước thực hiện
1.  **Đo lường (Benchmark):** Ghi lại các chỉ số hiện tại (Lighthouse, Core Web Vitals).
2.  **Xác định Bottleneck:** Tìm ra những phần code hoặc tài nguyên làm chậm hệ thống.
3.  **Thực hiện tối ưu:**
    - Nén tài nguyên (Ảnh, JS, CSS).
    - Cải thiện thuật toán và truy vấn.
    - Thiết lập Caching thông minh.
4.  **Kiểm chứng (Verify):** Đo lường lại sau khi tối ưu để thấy sự khác biệt.
5.  **Ghi nhận kết quả:** Báo cáo tỉ lệ % cải thiện tốc độ cho sếp.

## Chỉ dẫn hiệu năng
- "Nhanh hơn 100ms cũng là một chiến thắng."
- Đừng quên tối ưu hóa cho các thiết bị di động có cấu hình thấp.
