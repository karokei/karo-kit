# Karo Workflow: Generate Spec

## Giới thiệu
Quy trình chuyên nghiệp để tạo ra tài liệu đặc tả (Spec) từ yêu cầu của sếp.

## Các bước thực hiện
1.  **Thu thập yêu cầu:** Phỏng vấn sếp hoặc phân tích ý tưởng ban đầu.
2.  **Phác thảo cấu trúc:** Chia nhỏ các tính năng thành các module.
3.  **Viết chi tiết logic:** Xây dựng đặc tả cho từng module (Sử dụng kỹ năng `karo-logic-spec-writer`).
4.  **Thiết kế Flow:** Vẽ các sơ đồ Mermaid để minh họa.
5.  **Review Spec:** Kiểm tra tính logic và khả thi của toàn bộ Spec.
6.  **Bàn giao bản thảo:** Gửi Spec cho sếp xem và xác nhận.

## Lưu ý
- Spec là "kim chỉ nam" cho lập trình viên, nên cần cực kỳ rõ ràng.
